# Smart Home Networking Project (IoT)

This project demonstrates a **Smart Home system using IoT devices** designed and simulated in **Cisco Packet Tracer**.  
The goal of this project is to show how smart devices communicate over a network to automate daily home activities.

---

## Project Overview
In this project, different IoT devices are connected through a network to create an automated smart home environment.  
The system allows monitoring and controlling home devices using networking concepts.

- Devices connected using wired and wireless networks  
- IP addressing configured for all devices

## How to Run the Project
1. Install **Cisco Packet Tracer**
2. Open the `.pkt` file
3. Switch to **Simulation Mode**
4. Test and control IoT devices
